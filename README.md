## Getting Started

Run

In two different tabs, run
```bash
npx rbt run dev
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

Inspect the reboot backend at [http://localhost:9991/__/inspect](http://localhost:9991/__/inspect)
